# multi-label-text-classification
【Demo】多标签文本分类

*说明：数据集仅包含数十行示例数据*

# 如何使用

### 数据预处理

```shell
python prepare_data.py
```

### 训练和评估模型

```shell
python train.py
```

### 使用训练好的模型进行预测

```shell
python predict.py
```
